# Usage

## Run locally

```bash
make serve
```

Opens at `http://localhost:1313` with drafts and future posts enabled.

## Create a new post

```bash
make new POST="my-post-title"       # English
make new-ar POST="my-post-title"    # Arabic
```

## Build for production

```bash
make build
```

Output goes to `public/`.

## All commands

```bash
make help
```
